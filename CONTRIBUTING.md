# Contributing

欢迎提交规则修正、模板改进和资料来源更新。

## 可以贡献什么

- 新的官方资料链接。
- Gmail、Yahoo、Outlook 等平台规则更新。
- CAN-SPAM、GDPR、PECR、CASL、Spam Act 等合规说明修正。
- 更清楚的审核模板。
- 更保守、更可执行的风险阈值。
- 错别字、格式和表达改进。

## 不接受什么

- 绕过垃圾邮件过滤器的方法。
- 购买名单、洗名单后继续群发的技巧。
- 虚假暖箱、假打开、假回复、互助标记非垃圾等做法。
- 鼓励对退订、拒绝、投诉联系人继续发送的建议。
- 没有来源的“玄学送达率”规则。

## 提交建议

1. 先确认资料来源是官方或可信行业来源。
2. 修改文档或模板。
3. 在 PR 中说明为什么要改。
4. 如果是平台或法律规则变更，请附上来源链接和日期。

## 文档风格

- 中文为主。
- 表述要可执行。
- 优先写清楚“什么时候暂停”。
- 避免模糊话术，例如“适当控制”“注意一点”。
- 不承诺送达率结果。
