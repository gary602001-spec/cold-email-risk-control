## 使用原则

优先级如下：

1. 最新官方规则。
2. 发件平台当前要求。
3. 司法辖区法律和监管说明。
4. 行业组织最佳实践。
5. 冷邮件经验分享。

经验分享不能覆盖官方规则，不能作为绕过风控或继续骚扰收件人的理由。
